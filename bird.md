### Check Config

```
$ sudo birdc6 
BIRD 1.5.0 ready.
bird> configure check 
Reading configuration from /etc/bird/bird6.conf
Configuration OK
bird>
```

### Reload Config

```
$ sudo birdc6 
BIRD 1.5.0 ready.
bird> configure 
Reading configuration from /etc/bird/bird6.conf
Reconfigured
bird>
```
